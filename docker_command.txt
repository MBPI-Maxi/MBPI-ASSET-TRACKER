Procedure:
    install docker on windows 86-64

try and run docker desktop.

if there is an error make sure:
    - to run cmd on administrator then type 'wsl' there will be prompt please read to install the latest version of the wsl

docker should run now

Also make sure in the docker desktop application,
    - click on the setting on the docker
    - you should be able to see the resource tab
    - change the disk image location to a more preferred storage drive. Docker images consumes a large space

then in the source code run the code
    - docker compose up --build
